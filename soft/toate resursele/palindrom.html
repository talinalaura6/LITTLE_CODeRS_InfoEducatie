Numerele palindoame
Teorie
Cod
„Unde-s mulți puterea crește”! Haide să vedem ce număr obținem dacă adunăm toate cifrele. Există o mulțime de obiecte simetrice. Te-ai gândit vreodată la un număr care este simetric? Un număr palindrom este un număr al cărui invers coincide cu numărul inițial. Spre exemplu 123454321 este un palindrom. Urmărește pseudocodul alăturat iar apoi testează și algoritmii în C++ și Pascal. Aceștia utilizează același principiu ca algoritmul pentru inversul unui număr, însă creează inițial o copie a numărului pentru a putea verifica daca aceste corespunde cu oglinditul.

Pseudocod:
citeste n
copie ← n
cat timp a ≠ 0 executa
c ← n%10
inv ← inv + c
n ← n/2
daca copie = inv atunci
scrie ” Numarul este palindrom”
altfel scrie ”Numarul nu este palindrom”
Introduceti valoarea pentru:

a =

b =
